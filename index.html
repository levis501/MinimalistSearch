Minimal search app for android that launches quickly and opens soft keyboard immediately

Quickness is achieved by omitting features of the standard search

  • No search history maintained
  • No test of the internet connection is performed
  • No auto-complete
  • Minimal interface
  • No customizations (default search provider always used)

This app requires no special permissions, and is open sourced at https://github.com/levis501/MinimalistSearch.

https://play.google.com/store/apps/details?id=com.stevenlevis.minimalistsearch
